# wemos-weather-simple
Simple Weather Station

![sws.jpg](sws.jpg "Simple Weather Station")


## Configuration

- in ``src`` folder copy ``secrets_template.h`` to ``secrets.h`` and fill your own
  - Blynk API Token
  - WiFi SSID and Password
  
- ``pio run``
